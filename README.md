# NWS - a web server
Simple Nodejs web server
## Table of contents
* General Info
* Technologies
* Setup
## General Info
This project is a simple web server created with core modules of Nodejs only and linter ESLint.
## Technologies
* Nodejs v20.13.1
* ESLint v9.4.0
## Setup
To run this project, clone this project and run  
```bash
$ git clone
$ npm install  
$ node server.js
```
